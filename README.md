# Habit

Simple habit-tracking project (React frontend + optional Python backend).

## Setup

Prerequisites: Node.js (v16+), Python 3.8+ (optional)

Frontend:

```bash
npm install
npm start
```

Backend (optional):

```bash
python -m venv venv
venv\Scripts\activate    # Windows
pip install -r requirements.txt
python app.py
```

## Notes

- Frontend source: `src/`
- Static site: `public/`
- Python backend entry: `app.py`
